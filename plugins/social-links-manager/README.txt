A plugin to manage social media links and display them on your site using a shortcode. Links left empty will not be displayed.

== Description ==
Social Links Manager allows you to manage and display social media links through an admin page. Use the [social_links] shortcode to add these links anywhere on your site. Empty links will not display.

== Installation ==
1. Upload the `social-links-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Social Links Manager" in the admin menu to configure links.

== Frequently Asked Questions ==
= How do I use the shortcode? =
Simply add `[social_links]` to any page, post, or widget to display your social links.
